## 📖 About

Task Master is a comprehensive task management solution designed to help individuals and teams organize their work efficiently. Built with modern technologies and best practices, it provides a seamless experience for managing tasks, priorities, and deadlines.

## ✨ Features

- ✅ **Task Management** - Create, edit, delete, and organize tasks effortlessly
- 🎯 **Priority System** - Categorize tasks by priority levels (High, Medium, Low)
- 📅 **Due Dates** - Set and track deadlines for your tasks
- 🏷️ **Categories & Tags** - Organize tasks with custom categories
- 🔍 **Search & Filter** - Quickly find tasks with powerful search and filtering
- 📊 **Progress Tracking** - Monitor your productivity with visual statistics
- 🎨 **Modern UI** - Clean and intuitive user interface
- 💾 **Data Persistence** - Your tasks are saved automatically
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🌙 **Dark Mode** - Easy on the eyes with dark theme support

## 🚀 Installation

### Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14.0 or higher)
- npm or yarn
- Git

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/mahmoud-atta1/Task-Master.git
   cd Task-Master
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Run the application**
   ```bash
   npm start
   # or
   yarn start
   ```

5. **Open your browser**
   
   Navigate to `http://localhost:3000`

## 📝 Usage

### Creating a Task

1. Click on the **"Add Task"** button
2. Fill in the task details:
   - Task name
   - Description (optional)
   - Priority level
   - Due date
   - Category/Tags
3. Click **"Save"** to add the task

### Managing Tasks

- **Edit**: Click on a task to edit its details
- **Complete**: Check the checkbox to mark as complete
- **Delete**: Click the delete icon to remove a task
- **Filter**: Use the filter options to view specific tasks

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Create new task |
| `Ctrl + F` | Focus search |
| `Ctrl + K` | Open command palette |
